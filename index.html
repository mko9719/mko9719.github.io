    • Summary: Latitude vs. A Given Variable

    The purpose of this analysis was to understand how various weather factors change as you move closer to the equator. We pulled data for 500 different cities from the OpenWeatherMap API, and analyzed it through the use of Jupyter Notebook. The graphs were created via Matplotlib. Weather variables analyzed include the following:

    • Temperature
    • Cloudiness
    • Windspeed
    • Humidity
